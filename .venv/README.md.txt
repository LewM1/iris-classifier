# Iris Classifier (Decision Tree)

## Overview
End-to-end ML example from the Digital Marketing Mastery Module â†’ builds a decision-tree classifier on the classic Iris dataset using scikit-learn.

## Quick Start
To get started with the Iris Classifier project, follow these steps:

1. **Clone the Repository**:
bash
git clone https://github.com/LewM1/iris-classifier.git

2. **Navigate to the Project Directory**:
bash
cd iris-classifier

3. **Set Up a Virtual Environment**:
bash
python -m venv venv
source venv/bin/activate # On Windows use: venv\Scripts\activate

4. **Install Dependencies**:
bash
pip install -r requirements.txt

5. **Run the Training Script**:
bash
python src/train.py

## Dataset
The Iris dataset is a well-known dataset in machine learning, consisting of 150 samples of iris flowers, with features such as sepal length, sepal width, petal length, and petal width.

## Usage
After training the model, you can use it to make predictions on new data. Modify the `src/predict.py` script to input new iris flower measurements.
